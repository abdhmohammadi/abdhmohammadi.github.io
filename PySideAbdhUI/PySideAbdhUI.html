PySideAbdhUI

PySideAbdhUI is a modular and customizable UI component library built with PySide6. It is designed for creating professional and elegant desktop applications with consistent styles and enhanced functionality.

The package includes a variety of features, such as:

  • Custom Widgets: Includes components like StackedWidget, Notify, and StyledTableWidget for advanced UI design.
  • Theming Support: Bundled QSS (Qt Style Sheet) themes for easy application of custom styles.
  • Utility Functions: Provides tools like style loaders, path resolution, and notification systems to simplify development.
  • Bundled Resources: Comes with pre-packaged icons (.svg) and styles (.qss) that are auto-detected, even when bundled with PyInstaller.
  • Production-Ready: Fully compatible with PyInstaller and custom build scripts, making it easy to integrate into production environments.

This package is ideal for developers looking to build structured, visually appealing, and feature-rich desktop applications.
