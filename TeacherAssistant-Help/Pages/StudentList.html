Main Features:

  • How to grouping the students?
    • - When the ungrouped option is selected from the combo list, a list of students who have not yet been grouped will be displayed. In this case, a button will appear that can be used to group students, but before clicking on it, at least one person from the existing list must be selected. By clicking on this button, a list of defined groups is displayed to the user, the user selects a group from this list and clicks "OK". In this case, the selected students will be placed in this group.
