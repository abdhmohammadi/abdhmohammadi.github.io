Overview

This page has been developed for managing and interacting with educational resources. It allows the teacher to filter, view, and select resources, with the option to generate customized content such as quizzes, academic reports, and other educational documents. The content can be previewed in HTML format and exported as PDF.

Key Features

  • Filter and Display Resources: Filter resources based on a keyword like title of the textbook or part of leasson topic, and the system will display matching results in a grid view.

  • Template Selection for Content Generation: Choose from a list of predefined evaluation templates that allow customization of fields (e.g., student name, date, score).

  • Content Preview and Export: After selecting the desired resources and filling in the required information, the teacher can generate content and preview it as a PDF.

  • Student and Resource Distribution: Distribute resources to students with customizable distribution time and deadlines.

  • Bookmark: A bookmarking feature is provided to prevent unwanted distribution of assessment materials. This feature is especially useful when the teacher is preparing questions for official exams. Bookmarked items will not be sent without notification and approval.

Full Help

1. User Interface Overview

The main interface consists of the following key elements:

  • Filter Input: Type a keyword to filter resources.
  • Resource Grid: Displays educational resources as cards with details.
  • Template Selection: Choose from available templates for content generation.
  • Input Fields: Displayed depending on the template, for entering necessary information.
  • Generate Content Button: Click to generate the content after filling the fields.

2. Filtering Resources

To filter resources:

  • Enter a Keyword: Type a keyword in the filter box.
  • Filter Button: Click the filter button to apply the filter.
  • Clearing Filters: Click the clear filter button to view all resources.

3. Resource Selection and Management

  • Selecting a Card: Click on a resource card to select it. The details will appear in input fields.
  • Removing a Card: Click the "remove" option to delete a selected card.
  • Clearing All Cards: Click the "Clear All" button to remove all resources from the grid.

4. Generating Content

  • Template Selection: Choose the template for generating the educational content.
  • Input Fields: Fill in the necessary fields based on the selected template.
  • Generating Content: Click the "Generate Edu-content" button to create the content.
  • Previewing Content: A preview is shown in PDF format before final output.

5. Distribution Options

  • Selecting Students: Choose the students to distribute resources to from the list.
  • Setting Distribution Time and Deadline: Set the time and deadline for resource distribution.
  • Distribute: Click the "Distribute" button to send resources to the selected students.

6. Template-Specific Configurations

  • Classroom Quiz Template: Designed for quizzes with minimal input fields.
  • Formal Template: Used for more formal documents with extra fields for academic details.

7. Common Troubleshooting

  • Issue: No resources displayed after applying the filter.
    Solution: Ensure the filter keyword is correct, or clear the filter to see all resources.
  • Issue: Generated content does not appear as expected.
    Solution: Check that the correct template is selected and all required fields are filled.
