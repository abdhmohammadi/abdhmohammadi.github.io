PySideAbdhUI is a clean, reusable, and highly customizable UI component package built with PySide6/Python 13. It is ideal for building professional and elegant desktop applications, especially those needing consistent styles, structured windows, enhanced widgets, and notification systems.

KidMath is an engaging educational game designed to help elementary school students learn and practice basic arithmetic operations. The game focuses on addition and subtraction of two-digit numbers, making it perfect for first-grade students.

TeacherAssistant application is an educational management tool designed to streamline the daily tasks of teachers while enhancing the learning experience for students. It offers robust features for managing student data, designing and delivering customized learning activities, tracking progress, and producing formal assessments. Built with flexibility and user-friendly UI, This program can track the learning process in parallel through lesson activities and assessments, as well as behavioral events related to the learning process, which is specifically designed for use in the teaching job.
