PySideAbdhUI

Version History

A chronological record of all changes to the PySideAbdhUI framework, organized by semantic version. Each entry details new features, modifications, bug fixes, deprecations, and breaking changes across all five modules. Use the filter buttons to narrow down by release significance.

v1.0.0 Major
All Modules
2025-06-01

Initial stable release of the PySideAbdhUI framework. This version introduces the complete frameless window system with animated panels, a library of five reusable widgets, a card-based grid view with infinite scroll, a Chromium-based WYSIWYG rich text editor with KaTeX math rendering and PDF/DOCX import/export, a full resource and theme management pipeline with QSS template substitution and WCAG 2.0 contrast-based color generation, and a dedicated QSS stylesheet manager with accent color placeholder support and syntax validation.

+ Added
  • Window AbdhWindow class: frameless QMainWindow with custom titlebar, collapsible left/right side panels, size-lock system, and 8-direction edge/corner resize.
  • Window Size-lock mechanism (_lock_size / _unlock_size) to prevent Qt layout auto-growth when child widgets change sizeHint.
  • Window Left panel dual-mode system: pinned (persistent sidebar) and overlay (auto-closing icon strip), toggled via toggle_overlay().
  • Window Fade-in + slide animation on first show() using QPropertyAnimation with OutQuad easing.
  • Window Global eventFilter for auto-closing overlay panels on outside clicks and managing resize cursor shapes across the window.
  • Widgets StackedWidget: animated page navigation with go_back(), go_forward(), go_first(), go_last(), and configurable allow_same_types page duplication check.
  • Widgets Separator: horizontal or vertical divider line widget with configurable orientation and color.
  • Widgets Label: styled QLabel subclass with consistent framework theming and CSS class label.
  • Widgets RingProgress: circular progress indicator using QPropertyAnimation on a custom progress_value Qt property, rendered via paintEvent.
  • Widgets SearchBox: search input widget with magnifying glass icon and clear button, emitting textChanged signals.
  • CardGrid CardWidget: clickable card wrapper with selection state, hover highlighting, and clicked signal emission.
  • CardGrid CardGridView: scrollable grid with CRUD API (add_card, remove_card, get_card, clear), dynamic column reconfiguration based on container width, and infinite scroll via load_more signal.
  • CardGrid card_clicked, card_selection_changed, and load_more signals for event-driven card interaction patterns.
  • Utils get_resource_path(): resolves packaged resource paths via importlib.resources with pathlib.Path return type.
  • Utils get_icon(): loads SVG/PNG icons from the package's icons/ directory and returns a QIcon.
  • Utils get_styles_template(): reads QSS template files from the package's styles/ directory.
  • Utils ThemeManager class: JSON-persisted theme configuration with QSS template placeholder substitution, runtime widget property editing via regex, and apply_theme() to push stylesheets to QApplication.
  • Utils random_contrasting_hex(): generates random hex colors with WCAG 2.0-compliant contrast ratios against a given background color.
  • Styles QtStyleSheetManager class: regex-based QSS stylesheet lifecycle manager with file I/O, in-memory editing, and accent color placeholder system.
  • Styles add_property_to_widget(): adds or updates a CSS property within a widget's stylesheet block using regex matching.
  • Styles remove_stylesheet_by_name(): removes all stylesheet definitions for a given widget name, including pseudo-states (e.g., :hover).
  • Styles Accent color placeholder system: check_accent_color_placeholder(), extract_accent_color_value(), and replace_placeholder() for token-based QSS theming.
  • Styles validate_stylesheet(): static method that checks for unmatched braces, missing semicolons, and invalid characters in QSS source.
  • Editor RichTextEditor class: Chromium-based WYSIWYG rich text editor built on QWebEngineView with contenteditable HTML, multi-page layout, and document.title communication bridge.
  • Editor DebugPage class: custom QWebEnginePage subclass that mirrors JavaScript console messages to Python stdout.
  • Editor KaTeX math rendering: auto-detect and render LaTeX formulas in inline and display modes using bundled KaTeX resources.
  • Editor PDF import via Mozilla pdf.js (lazy-loaded) and PDF export via Chromium's built-in renderer with QPageLayout configuration.
  • Editor DOCX/ODT import/export via pypandoc with graceful degradation when Pandoc is not installed.
  • Editor Multi-page pagination engine: automatic overflow/underflow management with configurable page sizes (A4, Letter, B5, Edu-Item).
  • Editor Image insertion with drag-and-drop repositioning and 8-directional resize handles, plus page-by-page PNG capture export.
  • Editor Table editing with column resize, row/column insertion via context menu, and Tab key navigation.
  • Editor Full text formatting API: bold, italic, underline, strikethrough, font family/size, text/background color, paragraph alignment, and bidirectional text.
  • Editor Zoom controls, page navigation, margin editing dialog, and dirty tracking with isModified()/setClean().
⚙ Known Issues at Release
  • Widgets Typo: allow_same_tyoes in StackedWidget (should be allow_same_types). Functional impact: the attribute name is misspelled but consistent internally.
  • Widgets Nested loop variable shadowing in StackedWidget.add_page() causing incorrect duplicate page detection.
  • CardGrid CardGridView.clear() does not clear the internal cards dictionary, only removes widget items from the grid layout.
  • Editor Hardcoded Pandoc path (C:\Program Files\Abdh\pandoc\pandoc.exe) only works on Windows installations.
  • Editor QEventLoop blocking for synchronous JS results can freeze the UI thread on slow operations.
v0.4.0 Minor
StyleManagers
2025-05-18

Introduction of the QtStyleSheetManager module, providing a regex-based QSS stylesheet lifecycle manager separate from the higher-level ThemeManager in utils.py. This module operates purely on string data with no Qt widget dependencies, making it suitable for headless stylesheet preprocessing and build-pipeline integration. It features an accent color placeholder system, property-level editing, widget-level removal, and static syntax validation.

+ Added
  • Styles QtStyleSheetManager class with load_stylesheet(), save_stylesheet(), and saveas_stylesheet() for file-based QSS lifecycle management.
  • Styles add_property_to_widget() method for adding or updating individual CSS properties within a widget's stylesheet block using regex pattern matching.
  • Styles remove_stylesheet_by_name() method supporting removal of widget styles and their pseudo-states (e.g., QWidget:hover).
  • Styles Accent color placeholder system: check_accent_color_placeholder(), extract_accent_color_value(), and replace_placeholder() for token-based accent color substitution in QSS files.
  • Styles validate_stylesheet() static method for checking unmatched curly braces, missing semicolons, and invalid characters.
  • Styles add_stylesheet() method for appending raw stylesheet strings to the current in-memory stylesheet.
v0.3.0 Minor
Utils
2025-04-22

Added the ThemeManager class and color utility functions to utils.py, completing the resource-resolution and theming pipeline. The ThemeManager provides JSON-persisted theme configuration, QSS template placeholder substitution (for accent color and other tokens), and runtime widget property editing. The random_contrasting_hex() function generates colors that meet WCAG 2.0 contrast ratio requirements against a given background, useful for dynamic UI color generation.

+ Added
  • Utils ThemeManager class with load(), save(), apply_theme(), and edit_widget_property() for end-to-end theme management.
  • Utils get_color_roles() function returning a dictionary of standard Qt color roles mapped to human-readable names.
  • Utils random_contrasting_hex() function using HSL color space randomization with WCAG 2.0 relative luminance contrast ratio checking.
⚙ Known Issues at Release
  • Utils Unreachable fallback code in ThemeManager.load() after an early return statement when the theme file is not found.
  • Utils edit_widget_property() uses regex that may break on QSS with nested braces or complex selectors.
v0.2.0 Minor
CardGrid Widgets
2025-03-10

Introduction of the CardGridView module for scrollable card-based layouts, and the Widgets library providing five reusable UI components. The card grid supports dynamic column reconfiguration based on container width, CRUD operations on cards, selection state management, and infinite scroll. The widgets library includes an animated stacked widget, separator, styled label, ring progress indicator, and search box.

+ Added
  • CardGrid CardWidget class: click-able card with selection/hover states, clicked signal, and customizable inner widget hosting.
  • CardGrid CardGridView class: scroll area with QGridLayout, CRUD API, dynamic column recalculation on resize, and load_more infinite scroll signal.
  • CardGrid Three signals: card_clicked, card_selection_changed, and load_more for event-driven integration.
  • Widgets StackedWidget with animated page transitions and bidirectional navigation (back/forward history stacks).
  • Widgets Separator, Label, RingProgress, and SearchBox widgets.
⚙ Known Issues at Release
  • Widgets allow_same_tyoes typo in StackedWidget constructor parameter.
  • CardGrid Commented-out self.button.clicked.connect in CardWidget, suggesting an incomplete feature.
v0.1.0 Minor
Window Utils
2025-02-01

The first development release of PySideAbdhUI, establishing the core window shell and basic utility functions. The AbdhWindow class provides a fully frameless QMainWindow replacement with custom titlebar, collapsible side panels (pinned and overlay modes), a size-lock system to prevent Qt layout auto-growth, and manual 8-direction edge/corner resize. The utils module supplies resource path resolution and icon loading helpers using importlib.resources.

+ Added
  • Window AbdhWindow class with frameless window flags, translucent background, golden-ratio initial sizing, and centered positioning.
  • Window Custom titlebar with logo, navigation buttons (back/forward), title label, settings button, and window control buttons (minimize, maximize/restore, close).
  • Window Size-lock system (_lock_size / _unlock_size) pinning both minimumSize and maximumSize to prevent layout auto-growth.
  • Window Left panel dual-mode: pinned (persistent) and overlay (auto-closing with icon strip), toggled via toggle_overlay().
  • Window Right panel toggle with QPropertyAnimation width animation.
  • Window Maximize/restore animation using QPropertyAnimation on geometry with OutQuad easing at 175ms.
  • Window 8-direction edge/corner resize with RESIZE_MARGIN = 8px grab zone and appropriate cursor shapes via nativeEvent on Windows.
  • Window Global eventFilter installed on QApplication.instance() for overlay auto-close and cursor management.
  • Utils get_resource_path() function using importlib.resources for packaged resource resolution.
  • Utils get_icon() function loading SVG icons from the package's icons/ directory.
  • Utils get_styles_template() function for reading QSS template files.
